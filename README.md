# cc-plugin-template

cocos creator 3.x 插件开发模板

## # 初始化

-   使用 vscode 全局替换 cc-plugin-template 为你的插件名

## # 编译

-   安装编译 npm 包（已安装跳过）
    > npm i @muzzik/cc-plugin-cli
-   使用
    > cc-plugin-cli c -p '插件目录'

## # 注意

-   i18n 文件夹 必须在根目录且为 js 文件

## # 建议

-   css 编写参考：https://www.w3school.com.cn/css/index.asp
-   html 标签：想要啥直接百度
